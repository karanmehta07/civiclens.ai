# CivicLens AI 🗺️

**See it. Report it. Fix it.**

CivicLens AI is a crowdsourced civic infrastructure reporting platform. Citizens report issues like potholes, broken streetlights, garbage dumps, and water leaks by tapping a live map and uploading a photo — an AI model verifies the issue and estimates severity, the community votes to confirm and prioritize, and a Smart Priority Engine surfaces what needs fixing first.

Built solo end-to-end with Next.js 15, TypeScript, Tailwind CSS, and Leaflet.

## Features
- 📍 Interactive map with tap-to-report
- 🤖 AI photo verification (severity + confidence scoring)
- 🔁 Duplicate detection
- ⚡ Smart Priority Engine (severity + votes + age + hazard weight)
- 🔥 City-wide hazard heatmap
- 📊 Analytics dashboard
- 🏆 Gamified leaderboard with badges
- 🛠️ Admin console for issue lifecycle management

## Getting Started

npm install
